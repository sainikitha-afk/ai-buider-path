# 🚀 AI Builder Path 

## Overview

This repository contains all projects completed as part of the **AI Builder Path Internship**, covering the full lifecycle of modern AI systems — from prompt engineering to multi-agent workflows and deployment simulations.

Each lesson builds on the previous one, progressing from foundational concepts to advanced AI system design.

---

## Lessons Covered

---

### Lesson 1 — RAG Chatbot

Built a **Retrieval-Augmented Generation (RAG)** chatbot using local models.

**Key Features:**

* Document chunking and embedding
* FAISS vector database
* Context-based answering
* Local LLM integration (Ollama)

---

### Lesson 2 — Prompt Engineering

Optimized prompts using structured frameworks.

**Concepts Implemented:**

* CLEAR framework
* Chain-of-Thought prompting
* Prompt optimization techniques
* Prompt security awareness

---

### Lesson 3 — AI Agents

Designed agent-based systems with dynamic decision-making.

**Highlights:**

* Workflow vs Agent comparison
* Tool-based agent logic
* Query classification and routing
* API integration

---

### Lesson 4 — LangChain Research Agent

Built a multi-tool **LangChain-based research agent**.

**Tools Used:**

* RAG tool (technical dataset)
* MCP tool (internal project insights)
* Web search tool (industry trends)

**Key Concepts:**

* ReAct framework
* Tool calling
* Context-aware reasoning

---

### Lesson 5 — Multi-Agent System (LangGraph)

Implemented a **multi-agent support system** using LangGraph.

**Agents:**

* Supervisor Agent (routing)
* IT Agent (technical queries)
* Finance Agent (financial queries)

**Concepts:**

* Multi-agent coordination
* Supervisor-worker pattern
* Tool usage within agents

---

### Lesson 6 — Monitoring, Security & Deployment Simulation

#### Observability

* Implemented logging (LangFuse-style)
* Captured prompts, tool usage, and latency

#### Guardrails

* Input validation for sensitive queries
* Prevented unsafe or malicious requests

#### Evaluation

* Measured:

  * Correctness
  * Latency
  * Hallucination rate
  * Tool usage success

#### Web Crawler Agent (Bedrock Simulation)

* Built a web scraping agent
* Simulated AWS Bedrock architecture
* Implemented tool-based crawling and text extraction
